Ejercicio 1

Determina cuál de las siguientes variables son válidas y explica por qué:

$_myvar, $_7var, myvar, $myvar, $var7, $_element1, $house*5

Respuesta:

  • $_myvar es válida porque inicia con guión bajo.
  • $_7var es válida porque inicia con guión bajo.
  • myvar es inválida porque no tiene el signo de dolar ($).
  • $myvar es válida porque inicia con una letra.
  • $var7 es válida porque inicia con una letra.
  • $_element1 es válida porque inicia con guión bajo.
  • $house*5 es inválida porque el símbolo * no está permitido.

Ejercicio 2

Proporcionar los valores de $a, $b, $c como sigue:
$a = “ManejadorSQL”;
$b = 'MySQL’;
$c = &$a;

a. Ahora muestra el contenido de cada variable

b. Agrega al código actual las siguientes asignaciones:
$a = “PHP server”;
$b = &$a;

c. Vuelve a mostrar el contenido de cada uno

d. Describe en y muestra en la página obtenida qué ocurrió en el segundo bloque de asignaciones

Respuesta inciso a:

Valores iniciales:

a: ManejadorSQL
b: MySQL
c: ManejadorSQL

Cambio hecho del inciso b.

Respuesta inciso c.

Valores después del cambio:

a: PHP server
b: PHP server
c: PHP server

Respuesta inciso d.

Explicación:

La variable $c es una referencia a $a, por lo que al cambiar $a, también cambia $c. Luego, $b también se hace referencia a $a, por lo que todas apuntan al mismo valor.
